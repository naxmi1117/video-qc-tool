# Video QC Tool

This Streamlit app scans uploaded videos for:

- Frozen frames
- Video glitches
- Black frames
- Audio silence

It generates:

- QC report (CSV)
- Thumbnails of problem frames

## How to use

1. Deploy on Streamlit Cloud: https://share.streamlit.io
2. Drag & drop a video into the app
3. View QC report and thumbnails in the browser
